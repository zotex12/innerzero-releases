# InnerZero

A private AI assistant that runs entirely on your PC. Free forever.

## Download

**[Download InnerZero v0.1.0](https://github.com/zotex12/innerzero-releases/releases/latest/download/InnerZero-Setup-0.1.0.exe)** (~265 MB)

- Windows 10/11 64-bit
- 16GB+ RAM recommended
- GPU optional but recommended (NVIDIA 6GB+ VRAM)

## What is InnerZero?

- Local-first AI — all processing on your hardware
- Personal memory that learns from every conversation
- Voice and text interaction
- 30+ built-in tools
- No account, no subscription, no data collection
